REST Countries (React, Redux).

https://countries-test-app.now.sh

Used api: https://github.com/apilayer/restcountries

You can find countries by:
- country name;
- capital;
- region;
- subregion;
- regional block;
- time zone;
- calling code;
- country code;
- alpha-3-code;
- alpha-2-code;
- min and max population;
- min and max area.
